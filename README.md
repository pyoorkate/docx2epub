Doc2Epub is intended to take a very clean word document and spit out a functional but bare epub.
The reason it's designed this way is to attempt to minimize crud. A lot of epub generators produce
epubs that try and stick as close to the original document's formatting as possible.

But we want to keep file sizes as small as possible and make it easier to adjust the final document,
so instead of producing a complex epub that needs a lot of clean up, this produces a super basic epub
that you can edit in your epub editor of choice to bring in whatever styles you like.
